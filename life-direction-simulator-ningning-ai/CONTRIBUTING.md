# Contributing

感谢你愿意一起完善这个项目。

## 你可以如何参与

- 提出更好的提问方式
- 提交更清晰的输出模板
- 分享真实使用案例
- 补充边界说明和风险提醒
- 优化 README / FAQ / 示例文档
- 修正错别字、结构问题和表达不清的部分

## 提交建议前

请尽量说明：

1. 你使用的是哪个文件
2. 你遇到的问题是什么
3. 你建议如何修改
4. 修改后希望改善什么

## Pull Request 建议

- 尽量一次只解决一个问题
- 提交前请简单说明修改原因
- 如果涉及重大改动，请先开 Issue 讨论

## 讨论氛围

这个项目面向处于人生选择期的人。
欢迎真诚、具体、负责任的反馈。
请避免攻击性表达、羞辱式建议、绝对化论断。
